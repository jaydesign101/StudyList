기본

<label for="date1">날짜 입력 필드 :</label>
<input type="date" id="date1" >
  • HTML에서 날짜를 입력받기 위한 기본 요소
  • 달력 UI, 날짜를 선택하는 편리한 방법을 제공한다
  • 달력 아이콘을 클릭해야만 달력 UI 팝업이 호출된다
  • Safari 9 이전 브라우저는 미지원(달력 UI가 호출되지 않음)
  • 기본 달력 UI는 브라우저에서 자동으로 제공되는 기본 CSS 스타일로, CSS 커스터마이즈하는건 어렵다
최소&최대 날짜 지정

<label for="date2">날짜 입력 필드 :</label>
<input type="date" id="date2"
min="2025-01-01" max="2025-01-25" >
  • 날짜 선택 범위 시작 날짜 : min 속성 추가
  • 날짜 선택 범위 종료 날짜 : max 속성 추가
  • 옵션 값 포맷 : "YYYY-MM-DD"
초기값 지정

<label for="date3">날짜 입력 필드 :</label>
<input type="date" id="date3"
value="2025-01-14" >
  • 초기값 지정 : vlaue 속성 추가
  • 옵션 값 포맷 : "YYYY-MM-DD"
필수 입력 지정

<form>
   <label for="date4">날짜 입력 필드 :</label>
   <input type="date" id="date4" required >
   <button type="submit">제출</button>
</form>
  • 필수값 지정 : required 속성 추가
  • 제출 버튼 클릭시 "이 입력란을 작성하세요"라는 알림 팝업이 뜸
  • form 요소와 button 요소를 같이 사용해야 함
읽기 전용

<label for="date5">날짜 입력 필드 :</label>
<input type="date" id="date5" readonly >
  • 읽기 전용 지정 : readonly 속성 추가
  • 사용자가 날짜를 변경할 수 없다
비활성화 지정

<label for="date6">날짜 입력 필드 :</label>
<input type="date" id="date6" disabled >
  • 비활성화 지정 : disabled 속성 추가
  • 비활성화 처리 되며, 사용자는 날짜를 선택할 수 없다
